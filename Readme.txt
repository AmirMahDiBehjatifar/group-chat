
        Project Setup 

        softwares that you need for run this project : installed nodejs on your Os,mongodb ClientServer 
        1. first install dependencies 
            // Third party packages needed 
             npm i 

        2. make sure you have mongodb on your system 


        3. you can run the project 
         npm run devmod

         2.  Sign up with a account 
         // Note : we did not use validations so please try valid one 

         3. login to your account with signin button 

         4. then click to group chat
